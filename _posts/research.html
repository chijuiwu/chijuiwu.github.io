EagleSense is a system infrastructure for tracking people and devices in interactive spaces using real-time top-view depth-sensing. It enables HCI researchers to build new proxemic and multi-device interactions. Our results show that the system can recognize six different human postures and activities with 90% accuracy at 30 FPS.

Out of Sight is a toolkit that extends the Kinect v2 SDK for tracking occluded human joint positions. It combines the skeleton stream of multiple Kinects using a coordinate transformation algorithm to track the position of people during occlusion. It enables applications to "see through" obstructions. The tracking accuracy is within 16cm.

Recomputation is the process of creating a working environment for a piece of computer program, so that it can be referenced, maintained, and deployed in the future. Recompute is a web application which automates the process of recomputation. It takes a GitHub repository URL as input and creates a virtual machine, within which the program is rebuilt from source. Anyone can download the virtual machine and run the program from anywhere.
